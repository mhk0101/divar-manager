# divar-sheypor-automation

نرم‌افزار دسکتاپ فارسی برای مدیریت و اتوماسیون حساب‌های **دیوار** و **شیپور**.

این برنامه برای ورود و نگهداری حساب‌ها، ذخیره Session، انتخاب شهر و دسته‌بندی، استخراج آگهی، استخراج شماره تماس، ارسال پیام چت، زمان‌بندی عملیات و مدیریت خروجی‌ها ساخته شده است.

> این برنامه توسط هوش مصنوعی ساخته شده است.  
> آیدی تلگرام برنامه‌نویس: [@MB0001111](https://t.me/MB0001111)

## معرفی پروژه

`divar-sheypor-automation` یک برنامه دسکتاپ برای ویندوز است که با استفاده از Python، PySide6 و Playwright ساخته شده و به کاربر کمک می‌کند عملیات تکراری روی دیوار و شیپور را ساده‌تر و سریع‌تر انجام دهد.

کاربردهای اصلی برنامه:

- مدیریت چند حساب دیوار و شیپور
- ورود و ذخیره Session حساب‌ها
- اعتبارسنجی حساب‌های ذخیره‌شده
- انتخاب شهرها و دسته‌بندی‌ها
- استخراج آگهی‌های جدید
- استخراج شماره تماس فروشنده‌ها
- ارسال پیام چت سفارشی
- اجرای خودکار عملیات با زمان‌بندی
- ذخیره خروجی‌ها در فایل‌های قابل بررسی

## نصب و اجرای یک‌کلیکی

### نصب و اجرای اولیه

بعد از دانلود یا Clone کردن پروژه، روی فایل زیر دوبار کلیک کنید:

```bat
setup_and_run.bat
```

این فایل به‌صورت خودکار انجام می‌دهد:

1. بررسی نصب بودن Python
2. انتخاب نسخه مناسب Python
3. ساخت محیط مجازی `.venv`
4. بروزرسانی `pip`
5. نصب وابستگی‌های پروژه
6. نصب مرورگر Chromium مخصوص Playwright
7. بررسی فایل‌های شهر و دسته‌بندی
8. اجرای برنامه

### اجرای دفعات بعد

بعد از نصب اولیه، برای اجرای برنامه از فایل زیر استفاده کنید:

```bat
run.bat
```

اگر محیط مجازی وجود نداشته باشد، `run.bat` به‌صورت خودکار `setup_and_run.bat` را اجرا می‌کند.

## استخراج شماره تماس

### دیوار

- ورود به صفحه خود آگهی دیوار
- انتظار فقط برای بخش دکمه‌های آگهی، نه کل صفحه
- تشخیص دکمه‌های:
  - اطلاعات تماس
  - تماس امن
  - تماس ناشناس
  - چت
- کلیک روی دکمه اطلاعات تماس
- استخراج شماره از لینک `tel:`
- استخراج شماره از متن داخل باکس اطلاعات تماس
- تشخیص حالت‌های بدون شماره:
  - فقط چت
  - شماره مخفی
  - تماس ناشناس
  - اطلاعات تماس ثبت‌نشده

### شیپور

- ورود به صفحه خود آگهی شیپور
- انتظار برای دکمه تماس
- کلیک روی دکمه تماس
- فشردن Escape قبل و بعد از کلیک برای بستن پاپ‌آپ‌ها
- بستن مودال‌های مکان شیپور
- جلوگیری از باز شدن پروتکل‌های خارجی مثل `tel:`
- تلاش برای حل Captcha شیپور با EasyOCR
- امکان صبر برای ورود دستی Captcha در صورت نیاز
- استخراج شماره از `tel:` یا متن دکمه تماس
- وقفه تصادفی بین ۱ تا ۶ ثانیه برای رفتار طبیعی‌تر

## حالت هم‌زمان استخراج شماره و ارسال چت

در این حالت برنامه برای همان آگهی‌هایی که قرار است پیام چت ارسال کند، ابتدا شماره تماس را استخراج می‌کند.

ترتیب عملیات:

1. ورود به صفحه خود آگهی
2. استخراج شماره تماس
3. ذخیره نتیجه شماره در جدول
4. ذخیره شماره معتبر در Excel یا CSV
5. بررسی فعال بودن چت
6. ارسال پیام چت در صورت امکان
7. ذخیره وضعیت ارسال پیام

ویژگی‌ها:

- شماره و چت برای همان آگهی انجام می‌شود
- اگر چت فعال نباشد، برنامه بی‌دلیل وارد چت نمی‌شود
- اگر شماره موجود نباشد، نتیجه واضح در لاگ نوشته می‌شود
- اگر مرورگر بسته شود، عملیات ادامه پیدا نمی‌کند

Add Python to PATH

- اینترنت را بررسی کنید
- بهتر است Python 3.11 نصب باشد
- دوباره `setup_and_run.bat` را اجرا کنید

دستی اجرا کنید:

```bat
.venv\Scripts\python.exe -m playwright install chromium
```

فایل‌های Batch باید نسخه جدید و انگلیسی/ASCII باشند. اگر هنوز خطا دیدید، پوشه قبلی را حذف کنید و نسخه جدید پروژه را Extract کنید.
