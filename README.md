## 📌 معرفی پروژه

`divar-sheypor-automation` یک نرم‌افزار دسکتاپ فارسی برای مدیریت و اتوماسیون حساب‌های **دیوار** و **شیپور** است.

این برنامه برای کاربرانی ساخته شده که می‌خواهند عملیات تکراری مثل ورود به حساب، استخراج آگهی، استخراج شماره تماس، ارسال پیام چت و زمان‌بندی عملیات را ساده‌تر، سریع‌تر و قابل مدیریت‌تر انجام دهند.

### 🎯 کاربردهای اصلی

- مدیریت چند حساب دیوار و شیپور
- ذخیره و اعتبارسنجی Session حساب‌ها
- انتخاب شهر و دسته‌بندی
- استخراج آگهی‌های جدید
- استخراج شماره تماس فروشنده‌ها
- ارسال پیام چت سفارشی
- اجرای چندین زمان‌بندی خودکار
- ذخیره خروجی‌ها در JSON، CSV و Excel

## 🚀 نصب و اجرای یک‌کلیکی

> ساده‌ترین روش اجرا در ویندوز، استفاده از فایل آماده نصب و اجرا است.

### ✅ نصب و اجرای اولیه

بعد از دانلود یا Clone کردن پروژه، فقط روی فایل زیر دوبار کلیک کنید:

```bat
setup_and_run.bat
```

این فایل به‌صورت خودکار انجام می‌دهد:

| مرحله | توضیح |
|---|---|
| 1 | بررسی نصب بودن Python |
| 2 | انتخاب نسخه مناسب Python |
| 3 | ساخت محیط مجازی `.venv` |
| 4 | بروزرسانی `pip` |
| 5 | نصب وابستگی‌های پروژه |
| 6 | نصب Chromium مخصوص Playwright |
| 7 | بررسی فایل‌های شهر و دسته‌بندی |
| 8 | اجرای برنامه |

### ▶️ اجرای دفعات بعد

بعد از نصب اولیه، برای اجرا فقط این فایل را باز کنید:

```bat
run.bat
```

اگر `.venv` وجود نداشته باشد، `run.bat` به‌صورت خودکار `setup_and_run.bat` را اجرا می‌کند.

## 📞 استخراج شماره تماس

### دیوار

برای دیوار، برنامه وارد صفحه خود آگهی می‌شود و فقط منتظر لود شدن بخش دکمه‌ها می‌ماند، نه کل صفحه.

بخش موردنیاز:

```css
.post-actions
```

قابلیت‌ها:

- کلیک روی دکمه اطلاعات تماس
- استخراج شماره از لینک `tel:`
- استخراج شماره از متن داخل باکس اطلاعات تماس
- تشخیص شماره مخفی
- تشخیص فقط چت
- تشخیص تماس ناشناس
- تشخیص اطلاعات تماس ثبت‌نشده

### شیپور

برای شیپور، برنامه وارد صفحه آگهی می‌شود و دکمه تماس را بررسی می‌کند.

قابلیت‌ها:

- کلیک روی دکمه تماس
- استخراج شماره از `tel:` یا متن دکمه
- بستن مودال‌های مزاحم
- فشردن Escape قبل و بعد از کلیک
- تلاش برای حل Captcha با EasyOCR
- وقفه تصادفی بین ۱ تا ۶ ثانیه

## ⏰ زمان‌بندی چندتایی

برنامه از چندین زمان‌بندی همزمان پشتیبانی می‌کند.

ویژگی‌ها:

- زمان‌بندی برای شماره‌های مختلف
- زمان‌بندی برای دیوار و شیپور
- اجرای هر زمان‌بندی فقط یک‌بار در هر چرخه
- شروع چرخه بعدی بعد از اتمام همه زمان‌بندی‌ها
- توقف، ادامه، لغو یا حذف هر زمان‌بندی

مثال:

| شماره | پلتفرم | زمان‌بندی |
|---|---|---|
| 0912xxxxxxx | دیوار | ۵ دقیقه بعد |
| 0935xxxxxxx | شیپور | ۱۰ دقیقه بعد |
| 0902xxxxxxx | دیوار | ۳۰ دقیقه بعد |

بعد از اتمام همه، چرخه بعدی از ابتدا ساخته می‌شود.
